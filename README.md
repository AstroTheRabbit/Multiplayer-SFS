Anyone asking about the mod should be directed to this notice; I will likely ignore any messages/comments to me asking questions along the lines of "How is development going?", "How can I install the mod?", etc. I would love to know if you are considering working on a multiplayer mod yourself however, and would be fine with providing occasional help or tips about specific implementation details.
